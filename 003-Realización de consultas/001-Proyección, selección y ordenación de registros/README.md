Introducción
Estas consultas SQL trabajan con la tabla clientes para mostrar información de los registros almacenados. Se utilizan instrucciones SELECT para recuperar datos y ORDER BY para organizarlos según ciertos criterios, además de alias AS para mejorar la presentación de los resultados.

Aspectos técnicos
Las consultas seleccionan las columnas nombre, apellidos y edad, pudiendo renombrarlas con alias como “Nombre del cliente”. El uso de ORDER BY permite ordenar los resultados: por ejemplo, edad DESC ordena de mayor a menor edad, mientras que apellidos ASC lo hace alfabéticamente. También se pueden combinar varios criterios de ordenación o usar solo uno, como en el caso de ordenar únicamente por apellidos.

¿Para qué sirve?
Estas consultas permiten visualizar los datos de forma clara y organizada, lo cual es fundamental en aplicaciones reales. Ordenar por edad puede ayudar a analizar rangos de clientes, mientras que ordenar por apellidos facilita búsquedas o listados alfabéticos.

Conclusión
El uso de SELECT, alias y ORDER BY es básico pero muy potente en SQL. Permite adaptar la salida de los datos según las necesidades del usuario, mejorando la legibilidad y facilitando el análisis de la información almacenada en la base de datos.
